 Flight Search: Skyscanner’s Flight Search API provides live flight prices worldwide . (It covers
 52+ markets, 1,300+ airline/hotel partners .) Alternatives include Kiwi.com’s Tequila API or
 Amadeus API. These return flight options given origin, destination, dates. Pros: Up-to-date deals,
 global reach . Cons: Require registration/API keys, usage limits, and sometimes partner
 approvals.

Hotel Search: Skyscanner also offers a hotel search API. Other options include Google Hotels
 (via a paid API or scraping) or the Booking.com Affiliate API. These let you find hotels by location/
 dates. Pros: Current availability and pricing. Cons: Often limited to partners; may require extra
 negotiation. 
Weather Data: Services like OpenWeatherMap or WeatherAPI.com provide current weather
 and forecasts by city or coordinates. OpenWeatherMap offers free tier access with global
 coverage. Use it to give weather conditions for a destination/date. Pros: Easy to use, broad
 coverage. Cons: Strict rate limits on free plans; forecasts may lack very fine detail.

 Local Events: Ticketmaster Discovery API has 230K+ events worldwide (US, Canada, EU, etc.)
 . Eventbrite also has an events API. These let you find concerts, festivals, etc., near a city and
 date. Pros: Rich event info for recommendations. Cons: Mostly English-speaking markets; rate
 limits (e.g. 5000 calls/day ).
